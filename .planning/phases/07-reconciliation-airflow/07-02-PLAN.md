    **Task 1 — `detect_duplicates(ds=None)`** (per D-16):
    - Build window: `window_start = f"{ds} 00:00:00+00"`, `window_end = f"{ds} 23:59:59.999999+00"`
    - Create SQLAlchemy engine from `DATABASE_URL_SYNC`
    - Execute SQL:
      ```sql
      SELECT transaction_id, COUNT(*) as row_count
      FROM ledger_entries
      WHERE created_at >= :window_start AND created_at <= :window_end
      GROUP BY transaction_id
      HAVING COUNT(*) > 2
      ```
    - For each duplicate row, build a `ReconciliationMessage` dict:
      - `transaction_id` from query result
      - `discrepancy_type = "DUPLICATE_LEDGER"`
      - `stripe_payment_intent_id = None`
      - `internal_amount_cents = None` (multiple rows, ambiguous)
      - `stripe_amount_cents = None`
      - `diff_cents = None`
      - `currency = "USD"`
      - `merchant_id` — fetch from ledger_entries with a subquery: `SELECT DISTINCT merchant_id FROM ledger_entries WHERE transaction_id = :tid LIMIT 1`
      - `stripe_created_at = None`
      - `run_date = ds` (as date string)
    - Validate each dict through `ReconciliationMessage(**msg).model_dump(mode="json")`
    - Publish all via `ReconciliationProducer.publish_batch()`
    - Log: `structlog` with `reconciliation_duplicates_found`, count, run_date
    - Return count of duplicates found (for Airflow XCom)

    **Task 2 — `fetch_stripe_window(ds=None)`:**
    - Set `stripe.api_key = STRIPE_API_KEY`
    - Set `stripe.max_network_retries = 3` (per D-17)
    - Build window timestamps as Unix epoch ints:
      ```python
      window_start_dt = datetime.strptime(ds, "%Y-%m-%d").replace(tzinfo=timezone.utc)
      window_end_dt = window_start_dt + timedelta(days=1) - timedelta(microseconds=1)
      window_start_ts = int(window_start_dt.timestamp())
      window_end_ts = int(window_end_dt.timestamp())
      ```
    - Call `stripe.PaymentIntent.list(created={"gte": window_start_ts, "lte": window_end_ts}, limit=100)` with auto_paging_iter() (per D-14)
    - Filter to `pi.status == "succeeded"` only (per D-15)
    - Build dict keyed by `pi.id`: `{pi.id: {"amount": pi.amount_received, "currency": pi.currency, "created": pi.created, "metadata": pi.metadata}}`
    - Log: `stripe_window_fetched`, count, run_date
    - Return the dict (serialized via XCom)

      ```sql
      WHERE created_at >= :window_start AND created_at <= :window_end
        AND entry_type = 'DEBIT'
      ```
    - Build internal dict: `{row.transaction_id: {"amount_cents": row.amount_cents, "merchant_id": row.merchant_id, "currency": row.currency}}`
