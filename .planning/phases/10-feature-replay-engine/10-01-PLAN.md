**Module docstring** must include (per D-04, D-10):

- Description: Reconstructs ML features from historical PostgreSQL data (payment_state_log + ledger_entries)

  - hour_of_day: exact — from payment_state_log.created_at at INITIATED state
  - weekend_flag: exact — same timestamp
  - amount_cents_log: exact — from ledger_entries.amount_cents (DEBIT row)

  - tx_velocity_1m: approximated via SQL COUNT(*) OVER window, not streaming state
  - tx_velocity_5m: same approximation as tx_velocity_1m
  - amount_zscore: batch retrospective (mean/std per merchant over full batch), NOT Welford streaming

- A "No Label Column" note: Parquet does not include ground truth fraud labels; labeling strategy out of scope (per D-18)
- Usage: `python -m replay.feature_reconstruction [--start-date YYYY-MM-DD] [--end-date YYYY-MM-DD]`

**Function: `build_query(start_date, end_date)`**
Returns a SQL string (parameterized). The query must:
1. Select INITIATED rows from payment_state_log (where from_state IS NULL AND to_state = 'INITIATED') to get transaction start timestamps
2. Use SQL window functions for velocity (per D-07):
   ```sql
   COUNT(*) OVER (
       PARTITION BY psl.merchant_id
       ORDER BY psl.created_at
       RANGE INTERVAL '1 minute' PRECEDING
   ) AS tx_velocity_1m,
   COUNT(*) OVER (
       PARTITION BY psl.merchant_id
       ORDER BY psl.created_at
       RANGE INTERVAL '5 minutes' PRECEDING
   ) AS tx_velocity_5m
   ```
3. LEFT JOIN ledger_entries (entry_type = 'DEBIT') on transaction_id to get amount_cents and settled_at
4. Filter only transactions that have a DEBIT ledger entry (INNER JOIN or WHERE amount_cents IS NOT NULL) — only SETTLED transactions have training value
5. Apply optional WHERE clause for date range on psl.created_at using `%s` psycopg2 placeholders
6. Return tuple of (sql_string, params_list)

**Function: `compute_batch_zscore(df: pd.DataFrame) -> pd.Series`**
Per D-08: compute amount_zscore as batch retrospective per merchant.
- Group df by merchant_id
- For each merchant: mean = group mean of amount_cents, std = group std of amount_cents
- zscore = (amount_cents - mean) / std, use 0.0 if std == 0 or NaN
- Return a Series aligned with df index

**Function: `get_merchant_risk_scores(merchant_ids: list[str]) -> dict[str, float]`**
Per D-06: Try to read from Redis `merchant:risk:{merchant_id}`. Fall back to 0.5.
- Use redis.Redis.from_url(REDIS_URL) with decode_responses=True
- Wrap in try/except — if Redis unavailable, return 0.5 for all merchants
- Log warning if Redis unavailable

**Function: `reconstruct_features(conn, start_date=None, end_date=None) -> pd.DataFrame`**
Main logic:
1. Call build_query() and execute with psycopg2 cursor, load into pandas DataFrame
2. If empty result set, log warning and return empty DataFrame with correct columns
3. Compute deterministic features by iterating rows (per D-06):
   - hour_of_day: `compute_hour_of_day(row.created_at.isoformat())`
   - weekend_flag: `compute_weekend_flag(row.created_at.isoformat())`
   - amount_cents_log: `compute_amount_cents_log(row.amount_cents)`
4. Compute velocity features: already in SQL query result (tx_velocity_1m, tx_velocity_5m)
5. Compute amount_zscore: call compute_batch_zscore(df)
6. Get merchant_risk_score: call get_merchant_risk_scores() with unique merchant_ids
7. Set device_switch_flag = 0 for all rows (per D-09)
8. Build output DataFrame with METADATA_COLUMNS + FEATURE_COLUMNS
9. Cast feature columns to float64
10. Return DataFrame

**Function: `write_parquet(df: pd.DataFrame, output_dir: str) -> str`**
Per D-11, D-12:
- Create output_dir if not exists (os.makedirs)
- Filename: `features_{date.today().strftime('%Y%m%d')}.parquet`
- df.to_parquet(path, engine='pyarrow', index=False)
- Log the output path and row count
- Return the full file path

**Function: `parse_args(argv=None) -> argparse.Namespace`**
Per D-05:
- `--start-date`: optional, type=str, help="Start date YYYY-MM-DD (inclusive)"
- `--end-date`: optional, type=str, help="End date YYYY-MM-DD (inclusive)"
- Parse dates into date objects (or None if not provided)

**Function: `main(argv=None)`**
- Parse args
- structlog configuration (same as seed_demo_data.py pattern)
- Connect to PostgreSQL via psycopg2
- Call reconstruct_features()
- If empty, log and exit(0)
- Call write_parquet()
- Log summary: row count, output path, date range
- Close connection

**`if __name__ == "__main__": main()`**

Key constraints:

- Use psycopg2 directly (per D-03, matching seed_demo_data.py pattern)

- Do NOT write to Redis (per D-14)
- Do NOT write to BigQuery (per D-14, D-15 is stretch only)
