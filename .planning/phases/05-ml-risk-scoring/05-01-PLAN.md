    **ml/scorer.py** — `XGBoostScorer` class:
    ```python
    import os
    import sys
    import structlog
    import xgboost as xgb
    import numpy as np
    from models.ml_scoring import FeatureVector, RiskScore

    logger = structlog.get_logger(__name__)

    FEATURE_ORDER = [
        "tx_velocity_1m", "tx_velocity_5m", "amount_zscore",
        "merchant_risk_score", "device_switch_flag",
        "hour_of_day", "weekend_flag", "amount_cents_log",
    ]

    FEATURE_DEFAULTS = {
        "tx_velocity_1m": 10,
        "tx_velocity_5m": 50,
        "amount_zscore": 2.0,
        "merchant_risk_score": 1.0,
        "device_switch_flag": 1,
        "hour_of_day": 12,
        "weekend_flag": 0,
        "amount_cents_log": 0.0,
    }
    ```

    Constructor: `__init__(self, model_path: str)`:
    - If file does not exist at model_path: `logger.error("model_file_missing", path=model_path)` then `sys.exit(1)`
    - Load model: `self._booster = xgb.Booster(); self._booster.load_model(model_path)`
    - Log: `logger.info("xgboost_model_loaded", path=model_path)`
    - Store model in memory — never reload per inference.

    Method: `score(self, features: dict[str, float]) -> RiskScore`:
    - Build numpy array from features dict in FEATURE_ORDER
    - Create `xgb.DMatrix` from the array (shape 1x8, feature_names=FEATURE_ORDER)
    - `raw_score = self._booster.predict(dmatrix)[0]`
    - Clamp to [0, 1]: `risk_score = float(max(0.0, min(1.0, raw_score)))`
    - Return `RiskScore(risk_score=risk_score, is_high_risk=risk_score >= 0.7, manual_review=risk_score >= 0.85)`

    **ml/train.py** — Synthetic training script:
    - Comment at top: `# Run once to regenerate ml/models/model.ubj. Not part of the serving path.`
    - Generate 1000 synthetic samples using numpy:
      - Normal transactions: tx_velocity_1m ~ N(5, 2), tx_velocity_5m ~ N(20, 5), amount_zscore ~ N(0, 0.5), merchant_risk ~ U(0, 0.3), device_switch=0, hour_of_day ~ U(8,20), weekend_flag=0, amount_cents_log ~ N(7, 1)
      - Fraudulent transactions (30%): tx_velocity_1m ~ N(15, 3), tx_velocity_5m ~ N(80, 10), amount_zscore ~ N(2.5, 0.5), merchant_risk ~ U(0.5, 1.0), device_switch=1, hour_of_day ~ U(0,6), weekend_flag ~ Bernoulli(0.5), amount_cents_log ~ N(10, 1)
    - Train XGBoost binary classifier: `xgb.train({"objective": "binary:logistic", "max_depth": 4, "eta": 0.1, "eval_metric": "auc"}, dtrain, num_boost_round=50)`
    - Save to `ml/models/model.ubj` via `booster.save_model(path)`
    - Print AUC score to verify model quality
    - Create `ml/models/` directory if it doesn't exist

    After writing train.py, **run it** to generate model.ubj:

    cd payment-backend && python -m ml.train
