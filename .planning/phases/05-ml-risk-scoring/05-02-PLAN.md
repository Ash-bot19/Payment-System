    Method `_fetch_features(self, event_id: str) -> tuple[dict[str, float], bool]`:
    - Retry loop: 3 attempts, 50ms sleep between:
      ```python
      for attempt in range(FEATURE_RETRY_COUNT):
          try:
              raw = self._redis.hgetall(f"feat:{event_id}")
              if raw:
                  features = {k: float(v) for k, v in raw.items()}
                  return features, True  # (features, features_available=True)
          except redis.exceptions.TimeoutError:
              self._feature_timeout_counter.inc()
              logger.warning("redis_feature_timeout", event_id=event_id, attempt=attempt+1)
              return dict(FEATURE_DEFAULTS), False  # immediate fallback on timeout
          time.sleep(FEATURE_RETRY_DELAY_MS / 1000)
      # All retries exhausted — key not found
      self._feature_miss_counter.inc()
      logger.warning("redis_feature_miss", event_id=event_id, retries=FEATURE_RETRY_COUNT)
      return dict(FEATURE_DEFAULTS), False
      ```

    Method `_is_duplicate_scoring(self, event_id: str) -> bool` (per D-D5):
    - `SELECT 1 FROM payment_state_log WHERE event_id=:eid AND to_state='SCORING' LIMIT 1`
    - Returns True if row exists, False otherwise

    Method `_process_message(self, msg: Message)`:
    Processing order (per D-D3, D-D4):
    1. Decode JSON, deserialize as `ValidatedPaymentEvent.model_validate(raw_value)`
    2. Bind structlog context: event_id, merchant_id, partition, offset
    3. Idempotency guard: `if self._is_duplicate_scoring(event.event_id): log.info("duplicate_scoring_skip"); commit; return`
    4. Write `SCORING` state: `self._state_machine.write_transition(event.event_id, event.event_id, event.merchant_id, PaymentState.VALIDATED, PaymentState.SCORING)`
    5. Fetch features: `features, features_available = self._fetch_features(event.event_id)`
    6. Run inference: `risk_result = self._scorer.score(features)`
    7. If not features_available: override `risk_result` to force `manual_review=True` (belt-and-suspenders per D-C3)
    8. Determine final state: `AUTHORIZED` if risk_score < 0.7, else `FLAGGED`
    9. Write final state to DB FIRST (per D-D4): `self._state_machine.write_transition(event.event_id, event.event_id, event.merchant_id, PaymentState.SCORING, final_state)`
    10. Build `ScoredPaymentEvent` from event fields + risk_result fields + features_available
    11. Publish to `payment.transaction.scored`: `self._scored_producer.publish(event.event_id, scored_event.model_dump(mode="json"))`
    12. If `risk_result.is_high_risk`: also publish to `payment.alert.triggered`: `self._alert_producer.publish(event.event_id, scored_event.model_dump(mode="json"))`
    13. Increment Prometheus counters: `events_scored_total`, optionally `events_flagged_total`
    14. Commit Kafka offset: `self._consumer.store_offsets(msg); self._consumer.commit()`

    Exception handling in _process_message:
    - Wrap the entire body in try/except. On `redis.exceptions.TimeoutError` during feature fetch, the `_fetch_features` method already handles fallback. On any other exception (DB write failure, Kafka publish failure), let it bubble up to `run()` which crashes the consumer for Docker restart.

    **services/ml_service.py** — FastAPI app on port 8001:
    ```python
    import os
    import structlog
    from fastapi import FastAPI
    from prometheus_client import make_asgi_app, Counter

    from ml.scorer import XGBoostScorer
    from models.ml_scoring import FeatureVector, RiskScore

    logger = structlog.get_logger(__name__)

    app = FastAPI(title="ML Scoring Service", version="1.0.0")

    # Prometheus metrics
    metrics_app = make_asgi_app()
    app.mount("/metrics", metrics_app)
    score_requests_total = Counter("score_requests_total", "Total POST /score requests")

    # Load model once at startup (per D-A3)
    _scorer: XGBoostScorer | None = None

    @app.on_event("startup")
    def startup():
        global _scorer
        model_path = os.getenv("ML_MODEL_PATH", "ml/models/model.ubj")
        _scorer = XGBoostScorer(model_path)

    @app.get("/health")
    def health():
        return {"status": "ok", "model_loaded": _scorer is not None}

    @app.post("/score", response_model=RiskScore)
    def score(features: FeatureVector) -> RiskScore:
        score_requests_total.inc()
        result = _scorer.score(features.model_dump())
        logger.info("score_request", risk_score=result.risk_score, is_high_risk=result.is_high_risk)
        return result
    ```
